# Tetris Game

This project is a Tetris game built using React, Vite, and TailwindCSS. It aims to provide a fun, interactive way to play the classic game of Tetris in your browser, showcasing modern web development practices.

## Features
- Real-time gameplay
- Responsive design
- High score tracking

## Getting Started

### Prerequisites
- Node.js
- npm

### Installation
1. Clone the repository: `git clone https://github.com/KentKigen/redesigned-train.git`
2. Navigate into the project directory: `cd redesigned-train`
3. Install dependencies: `npm install`

### Running the Game
`npm run dev` to start the development server.

Enjoy playing Tetris!
